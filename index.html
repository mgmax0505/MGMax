How To Play

  • 1. Sign up with the details and login into the app.

  • 2. Add your points which you're going to use to play the games and those points will be added to the wallet.

  • 3. Select the game which you want to play from the games section.

  • 4. Once you select the game add points to the Numbers which you want to bet.

  • 5. You can see the result in history or on the main home.

  • 6. If you win, the points will be added to the wallet according to the game rates.

  • 7. You can withdraw points which are present in your wallet.
